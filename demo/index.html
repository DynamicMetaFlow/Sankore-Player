To use this demo, you need to use an HTTP server pointing to the parent directory using the /player/ URL of the server. This is necessary to server the sample UBW files used in these demos. You can hos the ubw files in a separate place from the player itself (for this you need to modify the path to the UBW file in the javascript launching the player).

The URL for your demo should be: http://host:port/player/demo/index.html

This demo contents two different ways to embed UBW files:

  • Showing an image preview of the first slide and clicking to launch the player in full screen: one file - two files
  • Showing the player in embedded mode
